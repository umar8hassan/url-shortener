# url-shortener
A URL Shortener app based on Flask

### 1. Build the app
```
docker-compose build
```

### 2. Start the app
```
docker-compose up -d
```

### 3. Open in browser to test
```
http://localhost:5000/docs
```

### 5. Use the user id generated from last step to pass as owner_id in URL endpoint
